* Setup
** Host
The host computer is a PC running any modern Linux distributions. I use Debian Trixie. The instructions in this book are based on Debian Trixie. You may need to adjust slightly if you use a different Linux distribution.
** QEMU
This driver runs on Linux kernel 2.6. The easiest way to develop it is creating a QEMU virtual machine that runs Debian 3.1 Sarge, which contains Linux kernel 2.6.8.
